# Pole Move Organizer

整理 pole dance 動作編碼清單，依數字由小到大排序，不更動任何文字內容。

## 功能

- 支援兩種輸入格式（自動偵測）
- 排序後自動驗證數量是否一致
- 提供 Claude Code skill 與 Google Sheets Apps Script 兩種使用方式

---

## 編碼格式說明

每個動作編碼由以下部分組成：

```
[前綴] [數字] [注記]
```

- **前綴**：`c`、`s`、`~` 或其組合（如 `~c`、`sc`），全部原樣保留，不省略
- **數字**：用於排序的基準
- **注記**：數字後方的任意文字（如 `未串`、`一半`、`to ayesha`、`(ball grip to chest)`），原樣保留

---

## 輸入格式

### 格式 A：逗號分隔

```
~c326, 328, 322 未串, s335, c360, c368 學完未串
```

### 格式 B：緊湊格式（無逗號）

動作編碼直接連寫，每個編碼以 `c+數字` 開頭：

```
sc325c329c335c337一半c339前c405/
```

展開後等同於：`sc325, c329, c335, c337一半, c339前, c405/`

---

## 使用方式

### 方式一：Claude Code Skill

放在 `~/.claude/skills/pole-move-organizer/` 後重啟 Claude Code 即可使用。

貼上編碼清單並說「排序」或「整理」即可觸發。

### 方式二：Google Sheets Apps Script

1. 開啟 Google Sheets
2. 選單 → **Extensions → Apps Script**
3. 將 `Code.gs` 的內容貼入編輯器，Cmd+S 存檔
4. 選函式 `onOpen`，按 **Run**，走完授權流程
5. 回到試算表，系統選單出現 **「Pole Moves」**

**使用**：點選有編碼的格子 → 「Pole Moves」→「排序選取的格子」

排序完成後會顯示驗證結果（數量是否一致）。

```
322 未串, ~c326, 328, 329, s335, 337, c360, c368 學完未串, 371 honey split
```

**格式 B 輸入**：
```
sc325c329c337一半c339前c405/
```

**輸出**：
```
sc325, c329, c337一半, c339前, c405/
```
