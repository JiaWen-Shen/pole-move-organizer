# Pole Move Organizer

整理 pole dance 動作編碼清單：移除 `c` 前綴，依數字由小到大排序，保留其他所有內容。

## 規則

### 前綴處理
- `c` 前綴：**移除**（`c326` → `326`，`c360` → `360`）
- `s` 前綴：**保留**（`s335` → `s335`）
- `~` 前綴：**保留**（`~326` → `~326`，`~c326` → `~326`）
- 大小寫：只處理小寫 `c`

### 排序
- 依每個 entry 的**數字部分**由小到大排序
- 數字相同的 entries 維持原有相對順序

### 保留
- 數字後方的所有注記文字一律保留（如「未串」、`to ayesha`、`(ball grip to chest)`、`學完未串`、`honey split`、`(~elbow)`）
- 空格、括號、英文注記，全部原樣保留

## 執行步驟

1. 將輸入以逗號分隔，逐一解析每個 entry
2. 對每個 entry：
   - 識別前綴（`~`、`c`、`s` 的組合）
   - 移除 `c`，保留其他前綴
   - 提取數字（用於排序）
   - 保留數字後方的所有注記文字
3. 依數字升冪排序
4. 以逗號分隔輸出

## 輸出後驗證

輸出後**必須確認**：
- 輸出的 entry 數量 = 輸入的 entry 數量
- 若數量不一致，找出差異並告知用戶

## 範例

**輸入**：
```
~c326, 328, 329, 322 未串, 327一半 缺jallegra, 331, 334, 337,338 to ayesha, 339 (ball grip to chest), 340, 341, 343, 349, s335,c360, c368 學完未串,337, 371 honey split, 372, 373(~elbow),396
```

**輸出**：
```
322 未串, ~326, 327一半 缺jallegra, 328, 329, 331, 334, s335, 337, 337, 338 to ayesha, 339 (ball grip to chest), 340, 341, 343, 349, 360, 368 學完未串, 371 honey split, 372, 373(~elbow), 396
```

**驗證**：輸入 22 個，輸出 22 個，一致。
